UnopinionatedAngularToolbox

This project was generated with Angular CLI version 1.5.0.

Development server

Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.

Running unit tests

Run ng test to execute the unit tests via Karma.

Build

Run npm run buildLib to build the library with ng-packagr.

Generate Documentation

Run npm run genDoc to generate the documentation in ./doc. This uses compoDoc to generate the documentation. Highly recommended for your Angular documentation needs.

Deploying to Github Pages

Currently no support for this through a simple command.
